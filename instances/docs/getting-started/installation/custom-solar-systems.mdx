# Installing Planet Packs

Planet Packs can be downloaded from the Spaceflight Simulator forum, which is available
[here](https://jmnet.one/sfs/forum/index.php?forums/planets-and-worlds.98/).

<Figure
  src="/gallery/PlanetPackSection.png"
  caption="Planet Pack section"
  alt="Planet Pack section"
  widthpercent="100"
></Figure>

This guide demonstrates the installation process using **Better Stock System** as an example.
Most Planet Packs include a downloadable attachment within their forum post, as shown below.

<Figure
  src="/gallery/ExamplePlanetPack.png"
  caption="Planet Pack attachment"
  alt="Planet Pack attachment"
  widthpercent="100"
></Figure>

Click on the attachment to download the Planet Pack.

After the download is complete, right-click the downloaded file and select **Extract All**.

<Figure
  src="/gallery/DownloadedPlanetPack.png"
  caption="Extracting the Planet Pack"
  alt="Extracting the Planet Pack"
  widthpercent="100"
></Figure>

Once extraction is complete, open the extracted folder. Inside, you will find a folder containing
the Planet Pack data. Copy this folder into the following directory:

Spaceflight Simulator Game/Spaceflight Simulator_Data/Custom Solar Systems


<Figure
  src="/gallery/CopyPlanetPack.png"
  caption="Copying the Planet Pack folder"
  alt="Copying the Planet Pack folder"
  widthpercent="100"
></Figure>

If the installation has been completed correctly, the **Custom Solar Systems** directory should
appear similar to the example below.

<Figure
  src="/gallery/PlanetPacksFolder.png"
  caption="Planet Packs folder structure"
  alt="Planet Packs folder structure"
  widthpercent="100"
></Figure>
